# Lock in

A minimal site you visit before focusing. Click **Lock in** → calming ambient sound and a breathing screen for ~1.5 minutes → then “You’re locked in.”

## Run locally

Open `index.html` in a browser, or use a local server:

```bash
npx serve .
```

Then visit `http://localhost:3000` (or the port shown).

## Deploy to GitHub Pages

1. Create a repo (e.g. `lockin`).
2. Push this folder to the repo.
3. In the repo: **Settings → Pages → Source**: Deploy from branch `main` (or `master`), folder `/ (root)`.
4. Your site will be at `https://<username>.github.io/lockin/`.

To use the custom domain `lockin.github.io`, the repo must be named **lockin** and belong to a user or org named **lockin**. Then enable Pages and add `lockin.github.io` in Settings → Pages → Custom domain.
